# Flyfish Word Editor Brand

## Official Name

- English: `Flyfish Word Editor`
- Chinese: `飞鱼 Word Editor`
- Repository: `flyfish-dev/word-editor`
- Product category: browser-native DOCX editor

## Language

- Simplified Chinese README: `README.md`
- English README: `README.en.md`
- Local WASM preview supports `?lang=zh-CN` and `?lang=en`, with an in-page language switch.

## Logo Assets

| Asset | Path | Usage |
| --- | --- | --- |
| Horizontal logo | `assets/brand/flyfish-word-editor-logo.svg` | README header, landing pages, documentation banners |
| Square mark | `assets/brand/flyfish-word-editor-mark.svg` | favicon, app icon, compact cards |
| Public beta badge | `assets/brand/flyfish-word-editor-public-beta-badge.svg` | social cards, release notes, public beta announcement |
| Social preview | `.github/social-preview.svg` | GitHub/social preview source artwork |

## Badge Language

- `public beta`: expanded repo
- `runtime`: sealed WASM
- `access`: localhost only
- `source maps`: not published
- `core payload`: sealed

## Brand Notes

- Use `Flyfish Word Editor` for public product references.
- Use `word-editor` only for the repository/package shorthand.
- Public beta messaging should clearly say `localhost-only` and `sealed WASM`.
- Do not describe this public repository as the full source repository.
